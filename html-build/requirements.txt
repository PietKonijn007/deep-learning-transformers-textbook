# Python dependencies for HTML conversion
# No external Python packages needed - uses only standard library

# System dependencies (install via package manager):
# - pdflatex (from MacTeX or TeX Live)
#   macOS: brew install --cask mactex
#   Linux: sudo apt-get install texlive-full
#
# - LaTeX packages (install via tlmgr):
#   sudo tlmgr install standalone
#
# - pdf2svg (for TikZ diagram conversion)
#   macOS: brew install pdf2svg
#   Linux: sudo apt-get install pdf2svg
#
# Alternative to pdf2svg:
# - ImageMagick (convert command)
#   macOS: brew install imagemagick
#   Linux: sudo apt-get install imagemagick
