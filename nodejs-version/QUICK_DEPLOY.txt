┌──────────────────────────────────────────────────────────────┐
│  METHOD 1: One Command (Easiest)                             │
└──────────────────────────────────────────────────────────────┘

  cd nodejs-version
  ./deploy-vercel.sh

  → Choose option 1 (preview) or 2 (production)
  → Done! 🎉

┌──────────────────────────────────────────────────────────────┐
│  METHOD 2: Manual CLI (5 minutes)                            │
└──────────────────────────────────────────────────────────────┘

  1. npm install -g vercel
  2. vercel login
  3. cd nodejs-version
  4. vercel
  5. vercel --prod

  → Your app is live! 🌐

┌──────────────────────────────────────────────────────────────┐
│  METHOD 3: Dashboard (No CLI)                                │
└──────────────────────────────────────────────────────────────┘

  1. Push code to GitHub
  2. Go to vercel.com/new
  3. Import repository
  4. Set root: nodejs-version
  5. Click Deploy

  → Automatic deployments on push! 🔄
